# ①課題名
meditation.still
（自然音と映像で心を静める瞑想用Webアプリ）

## ②課題内容（どんな作品か）
自然の音と映像を組み合わせた瞑想用Webアプリです。

深海・焚き火・雨・森・川といった自然環境をテーマに、
映像と環境音がシームレスに切り替わることで、作業前・休憩・就寝前など、日常の中で心を整える時間を提供します。

操作音やUIも主張しすぎない設計にし、「使っていることを忘れられる」ことを重視しました。

## ⑤工夫した点・こだわった点
・映像・音声のクロスフェード切り替え
シーン切替時に急な変化が起きないよう、映像・音声ともにフェード処理を実装しました。

・Web Audio API を用いた音量設計
BGM（自然音）とUI音（クリック音）を分離し、瞑想の妨げにならない音量バランスを細かく調整しています。

・自然音に限定したラインナップ
雷など刺激の強い音はあえて除外し、長時間聴いても疲れにくい自然音のみを採用しました。

・UIの主張を抑えた設計
一定時間操作がないとUIがフェードアウトする仕様にし、映像と音に集中できる体験を意識しました。
